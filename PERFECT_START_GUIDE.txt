Problem: "Port already in use"
Solution:
  1. Close all terminal windows
  2. Run RUN.bat again
  3. If still error, restart computer

Problem: "Module not found"
Solution:
  1. Make sure you're in the correct directory
  2. Make sure virtual environment is activated
  3. Run: pip install -r requirements.txt

Problem: "npm start failed"
Solution:
  1. Make sure you're in frontend directory
  2. Run: npm install
  3. Run: npm start

Problem: Backend not starting
Solution:
  1. Check if port 8000 is available
  2. Make sure .env file exists in backend folder
  3. Run: python manage.py migrate

Problem: Frontend not starting
Solution:
  1. Check if port 3000 is available
  2. Make sure node_modules exists
  3. Run: npm install
